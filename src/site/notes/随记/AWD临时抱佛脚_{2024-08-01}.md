# 基本防守策略

## 改用户密码和服务密码

**修改密码**：
```shell
passwd (用户)
```


**删除后门用户**：
```shell
//列出所有可登录用户
cat /etc/passwd | grep bash

//删除用户
userdel -r [用户名]
```


---

**修改mysql数据库密码**
```sql
update mysql.user set password=password('密码') where user='root';
```


**删除mysql匿名用户**
```sql
delete from mysql.user where user=' ';
```

**刷新配置**
```sql
flush privileges;
```
