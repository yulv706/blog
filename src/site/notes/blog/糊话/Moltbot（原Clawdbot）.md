说实话，现在AI发展真是太快了，过几天就会火一个新技术，对于我这种保守派懒人来说感觉不是很友好
但说实话，我还是有一点点好奇的，所以我就来以我**短浅又刻薄**的眼光来看一下这个**Moltbot**


# 是个啥

以我的理解，他就是个

**可以添加到日常聊天应用中的AI助手**

先来看看别人宣称的优点吧：
+ **运行在你自己的设备上**
	+ emmm，这个最大的用处就是就是说增加了数据隐私的保护，因为所有的对话数据都是保存在本地而非哪些AI服务商
	+ 但我说实话，这么功能我只能给到**NPC**，因为我认为这个功能太水了，就像是没啥功能点硬说的一样。首先，如果你要借用那些主流AI，肯定要把自己的数据传到那些AI服务商手里的吧（应该是，我并不是很确定Moltbot的逻辑），那么那些AI服务商想要保存一下你的输入并非难事，只是他们想不想的问题。而且这些对话数据即使不能被那些AI服务商保存，但是也是保存到你在那些你使用对话的软件中啊。
+ **通过日常消息应用交互**
	+ 这个功能我认为就是他这个项目最大的卖点了，可以集成到你的日常应用中，就像你在WX中添加了一个AI好友，你有啥问题直接在WX问他就行，还能帮你处理一些WX中的数据之类的
	+ 但我又想说实话，这个功能我只能给到**人上人**。首先确实有点创新吧，想象一下，自己在飞书或者WX聊着天，突然想问AI问题了，那么你不用再切换应用就能和AI实现对话，是在是太方便了，吗？
	+ 说实话，这功能对于我来说是一个可有可无的功能，因为我平时和AI对话并不多，只有在工作学习的时候才会高强度使用AI，那么那时候反而不怎么使用WX之类的了。其次你把AI移到那么日常应用中，AI的性能会发生什么变化呢，我感觉大概率是不如你在那些官方网页上使用AI来的强的。
	+ 这个功能对我最大的期待就是它可以无感的帮我处理那些应用中的事情，但是，这个功能对于我来说还是接近可有可无的程度，先不说他这个功能真的能做的这么好（我认为他要能真正做好这个功能，必然得让那些应用去主动做一下适配，但我感觉很难的啊），对于现在的我来说，真没有要让AI帮我处理消息的需求。
	+ 看了一下，主要是在 WhatsApp、Telegram、Slack、Discord里面用的，这不巧了吗，我都不怎么用，但是我看有人在飞书里面也搭建起来了，但是我用了一下，没意思。
+ **支持多个 AI 模型**
	+ 就是能让你接入Claude、GPT、Gemini 等主流 LLM
	+ 不多说，**拉完了**，毫无创新，没意思



# 我的评价

总体评价可以给到NPC
一个无聊又鸡肋的AI产品，有一点点创新和实用，但对我毫无吸引力。


以上仅代表个人看法，纯主观，毫无客观可言
